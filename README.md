# 🛍️ Mini E-Commerce Website (Django)

A modern and responsive **Clothing E-Commerce Web Application** built using Django.

---

## 🚀 Features

- 🛒 Product Listing Page
- 🔍 Search Functionality
- 🏷️ Category Filtering (Men/Women)
- 🖼️ Product Images Upload
- 📦 Order System
- 👨‍💼 Admin Panel (Add/Edit Products)
- 💻 Clean UI (Inspired by Myntra)

---

## 🛠️ Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite
